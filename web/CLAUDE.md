# フロントエンド

## 技術スタック

- React 19
- TypeScript
- Vite
- TanStack Router
- TanStack Query
- Tailwind CSS 4

## ディレクトリ構成

- `web/src/routes`: TanStack Routerによるファイルベースルーティング
- `web/src/components`: 共通コンポーネント
- `web/src/api`: バックエンドAPIクライアント

## コマンド

```bash
# 開発サーバー起動
pnpm dev

# ビルド
pnpm build

# Lint
pnpm lint

# プレビュー
pnpm preview
```
